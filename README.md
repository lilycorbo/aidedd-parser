# aidedd-parser

Utilities for fetching and parsing AideDD content

## Installation

`npm install`

## Usage

### Spells

`npm run spells` will generate the JSON representation of all standard 2024 spells, and write it to `out/spells.json`

### Feats

`npm run feats` will generate the JSON representation of all standard 2024 feats, and write it to `out/feats.json`
